# unity-material-cleaner

Unityでマテリアルのシェーダを切り替えた時に、前のシェーダのプロパティがそのまま残ってしまうため、不要なプロパティを削除するツールです。
  
詳しい解説は[**こちら**](https://amagamina.jp/material-cleaner)
